# Plataforma de Capacitação Educacional

Projeto de desenvolvimento de um portal web para a formação continuada dos profissionais da educação da Secretaria Municipal de Educação de Presidente Kennedy - ES.

Sobre o Projeto
Este repositório contém o protótipo funcional desenvolvido como requisito para a avaliação semestral da disciplina de Estágio Supervisionado: Atividade Prática Profissional, do curso de Análise e Desenvolvimento de Sistemas.

Funcionalidades
- Interface para navegação do catálogo de cursos de capacitação.
- Sistema de Autenticação (Login, Cadastro e Recuperação de Senha).
- Painel de Administração para cadastro de novos cursos.
- Emissão de Certificado ao término do curso.

Tecnologias Utilizadas
- HTML5: Estruturação semântica.
- CSS3: Estilização da interface.
- JavaScript: Lógica de interação no front-end.

Credenciais de Acesso (Teste)

- Login: admin
